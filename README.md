# Owen's Cheat Menu

A feature-rich mod menu for Retrowave that provides money tools, vehicle customization options, unlocks, upgrades, stat editing, and various utility functions. Designed for players who want to experiment with the game, customize vehicles, unlock content, or simply have fun without grinding.

## Features

* abilty to drag menu
* ability to turn off and turn on crashes or losses
* Auto-Saves turned on options
* Set combo in drive to max
* set bomb health to max
* add 100k to score in drive

### Money

* Set currency to $999,999
* Set currency to $9,999,999
* Add $10,000
* Currency multipliers
* Auto-Money (keeps your money at 9M no matter the purchases)

### Vehicle Upgrades

* Max Speed
* Max Handling
* Max Brakes
* Unlock NOS
* Unlock Turbo
* Unlock Siren
* Max all upgrades on selected vehicles
* Max all upgrades on every vehicle
* ability to increase vehicle speed

### Vehicle Unlocks

* Unlock all cars
* Unlock all colors
* Unlock all wheels
* Unlock all License Plates

### Statistics

* Set wins for all vehicles
* Modify total games played
* Modify win multipliers
* Various progression-related options
* set traffic difficulty to Peacful, Easy, Normal, Hard, Mega

## Installation

1. Install MelonLoader v0.4.3.0.
2. Launch Retrowave once after installation.
3. Close the game.
4. Open your Retrowave installation directory.
5. Open the Mods folder created by MelonLoader.
6. Place the mod DLL into the Mods folder.
7. Launch Retrowave.
8. Press F1 to open or close the menu.

If the Mods folder does not appear, verify that MelonLoader was installed correctly and launch the game once before trying again.
