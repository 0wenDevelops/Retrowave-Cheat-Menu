# Support

This repository is maintained as a personal modding project. Support is handled through GitHub Issues when enough information is provided to reproduce the problem.

## Before opening an issue

Please check the following first:

- You are using the latest release from this repository.
- You installed the supported loader version listed in the README.
- You placed the compiled DLL in the correct folder.
- You launched the game once after installing the loader.
- You backed up your save data before testing.
- You checked the README and release notes.

## Good bug reports include

- Mod version.
- Game version.
- Loader version.
- A clear description of the issue.
- Exact steps to reproduce it.
- Logs or screenshots when possible.

## Scope

Supported:

- Offline single-player usage.
- Installation questions.
- Reproducible bugs.
- Documentation corrections.
- Reasonable feature suggestions.

Not supported:

- Multiplayer usage.
- Public leaderboard manipulation.
- Competitive or online use.
- Modified third-party builds from unofficial sources.
- Issues caused by mixing files from multiple releases.
