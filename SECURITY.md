# Security Policy

## Supported versions

Only the latest public release is supported.

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Older releases | Limited |

## Reporting a security concern

If you believe a release file, dependency, or repository asset has a security issue, please open a GitHub issue with clear details.

Include:

- A description of the concern.
- The affected file or release version.
- Any scanner results or logs.
- Steps to reproduce or verify the issue.

## Download safety

Users should only download builds from this repository's official Releases page.

Recommended precautions:

- Download only from the official repository.
- Avoid reuploaded files from unknown sources.
- Scan downloads before running them.
- Keep backups of save data before installing mods.

## Project scope

This project is provided for offline single-player modding. Online, multiplayer, competitive, or leaderboard-focused use is not supported.
